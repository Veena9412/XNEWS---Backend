# XNEWS---Backend
XNEWS is a Java-based project that demonstrates REST API development, database integration, and modular design for a simple news management system. It allows users to create, read, update, and delete news articles, while practicing OOP principles, exception handling, and test-driven development.
Features:
1. CRUD operations for news articles (title, author, content, timestamp)
2. RESTful APIs built with Spring Boot
3. Database persistence using JPA/Hibernate
4. Input validation and error handling
5. Unit testing with JUnit
